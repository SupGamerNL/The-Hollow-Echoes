About the game

The Hollow Echoes blends tense real-time survival windows with deliberate point-and-click investigation. Manage cameras, conserve power, and piece together fractured memory fragments to uncover the facility's true purpose.

  • Genres: Horror, Sci-fi, Mystery
  • Core loop: Survive → Investigate → Recover memories → Choose
  • Perspective: First-person survival + fixed investigation scenes

Gameplay systems

  1. Camera & power system — toggle cameras, listen for mechanical cues, balance door/light usage with limited energy.
  2. Enemies — animatronic stalkers and psychological Echo events with unique behaviors.
  3. Point-and-click scenes — click hotspots, combine items, and unlock memory fragments between nights.

Design goals

Minimal UI, atmospheric lighting, and tight loops that deliver suspense. Diegetic interfaces (tablet/camera) whenever possible to keep HUD clutter low.

Minimum Viable Prototype

  • Single night loop with one animatronic (patrol + chase)
  • 3-camera network with power consumption
  • One investigation room + 2-item puzzle
  • One memory fragment and one Echo hallucination event

Prototype built in Unity using Visual Scripting (recommended). Renderer: URP. Use Cinemachine for camera transitions.
